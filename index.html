Welcome

Here are the materials we used on the FFC Game Dev Workshop. It's focues heavily on the CreateJS suite as well as some game development fundamentals.

  • Presentation : same presentation I gave with a few mods-- tweaks on the code and additional explanations. I added more challenges in the end.
  • Game Skeleton : the bare structure of the game. Use the presentation as guide.
  • Finished Game : the finished product

Reminder

Run the game on an http server (apache, WEBrick etc) to avoid asset loading errors
